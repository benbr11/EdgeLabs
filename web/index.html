⚽ World Cup 2026

How this works & where the data comes from

An independent data model that simulates each match many times to estimate win/draw odds, expected goals, and likely scorelines. Built for insight and fun — not betting advice, and never a certainty (football is famously unpredictable).

The method

Every team gets an attack and defense rating. For a match, those become each side's expected goals, and a Poisson model with the Dixon-Coles correction (a standard football-analytics method) turns them into probabilities and scorelines. Knockouts add extra time + penalties; weaker teams are modelled as more volatile.

Where the data comes from

  • ⚽ Match results — every international since 1872, weighted toward recent games.
  • 🎯 Expected goals (xG) — shot-quality data from StatsBomb (recent World Cups, Euros, Copa América, AFCON) plus this tournament's xG. Current World Cup games count extra.
  • 💰 Squad value — each team's player market value (Transfermarkt) — the talent on the pitch.
  • 📊 Elo & FIFA ranking — long-run strength and the official ranking.
  • 🌎 Match context — stadium altitude / heat / roof, rest days, and whether a team is already through or must win (an effect measured from 300+ past matches).
  • 🔄 Auto-updated twice a day with the latest results — data through —.

What each tab shows

  • Predict — the match estimate, expected goals, and most likely scorelines.
  • Groups — live standings and each team's chance to reach the knockouts.
  • Fixtures — every upcoming match, auto-predicted.
  • Title odds — a projection of who wins it all (a rating-seeded simulation, not the official bracket).

Backtested on 370+ past matches and competitive with bookmaker accuracy. The model is independent of betting markets — it only uses odds to flag where it sees value.

🏒 NHL

How the NHL model works

Each team gets an attack and defense rating, blended from several hockey metrics, then a Poisson model turns them into win odds, totals, and the puck line. Built for insight — not betting advice.

The metrics (what actually drives a game)

  • 🥅 Shot-quality expected goals (xG) — every shot's x/y location → chance quality created (offense) and allowed (defense), from MoneyPuck.
  • 🧤 Goaltending (GSAx) — goals saved above expected; stopping high-danger shots at a higher clip.
  • 📊 Goals model + Elo — results, margin, and long-run strength (recency-weighted).
  • ⚡ Special teams — power play and penalty kill.

Per-game factors you can set

  • Starting goalie — the single biggest swing in a hockey game.
  • Rest / back-to-back, injuries, and home ice.

Self-updating: pulls the latest games and shots daily and rolls forward every season automatically. Data through —.

League

How the league model works

Each club gets an attack and defense rating from recent results, then a Poisson + Dixon-Coles model turns a matchup into win/draw odds, expected goals, both-teams-to-score, clean sheets and likely scorelines — the same engine as the World Cup model.

  • ⚽ Results — last 3 seasons (football-data.co.uk), weighted toward recent games.
  • 📊 Elo stabiliser + goal-level calibration so totals stay realistic.
  • 🔄 Self-updating — re-pulls weekly and auto-rolls to each new season.

Independent of betting markets — built to find edges, not match the book.
